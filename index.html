About FDO Toolbox

FDO Toolbox is an multi-purpose geospatial tool to create, analyse, process and manage spatial data. It is written in C# and uses the Feature Data Objects (FDO) API

FDO Toolbox (1.1 and newer) requires .net Framework 4.0. Older releases require .net Framework 2.0 SP2

Disclaimer: Know what you are doing before trying this on production-quality geospatial data.
